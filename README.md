## 什么类型的网站可以添加

必须同时满足：
 1. 非营利性、无商业广告、无木马植入，不违反中华人民共和国网络安全法。
 2. 需要安全连接HTTPS，不需备案，有自己的域名（包括*.github.io等二级域名）。
 3. 建站需半年或有十篇原创文章以上，仅限博客，类型不限，内容积极向上，保持更新。
 4. 无违反国家相关法律法规和道德底线的内容，不涉反动言论和政治敏感话题，不传播色情淫秽，邪教组织等内容。
 5. 申请前请确保贵站已经添加本站。
 6. 看我心情。
